Free resources

Strategy and Getting started

  • Digital transformation - This paper from Cisco proposes an useful framework for developing a small businesses digital transformation strategy
  • Elements of AI- This short course provides a free, non-technical, overview of AI, what it can and can't deliver

Programming

  • Python first steps- This course from Microsoft is a good place to get started learning python
  • Python DataScience Handbook- This free book provides a fantastic introduction to the main python libraries used in developing an analytics capability
  • Flask Mega Tutorial - This free tutorial provides an excellent introduction to building web apps using python
  • Machine Learning Mastery - Loads of resources for starting to devlop your own machine learning

Maths

  • Khan Academy - Linear Algebra, Statistics and Calculus are key to developing machine learning capabilities. If you need to brush up these skills Khan Academy is a great resource
