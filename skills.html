Digital Skills and Resources

In this section we have currated a list of resources to help you on your digital transformation journey.

  • Free Resources
  • Subscriptions sites
  • Kids code club - Developing the next generation of digital skills

Free resources

Strategy

  • Digital transformation - This paper from Cisco proposes an useful framework for developing a small businesses digital transformation strategy
  • Elements of AI- This short course provides a free, non-technical, overview of AI, covering what it can and can't deliver

Programming

  • Python first steps- This course from Microsoft is a good place to get started learning python
  • Python DataScience Handbook- This free book provides a fantastic introduction to the main python analytical libraries
  • Flask Mega Tutorial - This free tutorial provides an excellent introduction to building web apps using python
  • Machine Learning Mastery - A great source of ideas when developing your own machine learning applications

Maths

  • Khan Academy - Linear Algebra, Statistics and Calculus are key to developing machine learning capabilities. If you need to brush up these skills Khan Academy is a great resource
